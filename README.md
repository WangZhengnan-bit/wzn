# 智慧图书管理系统

一个基于 **C++** 和 **控制台交互** 的简易图书馆管理系统，支持图书、用户、借还记录等基础功能。

## 功能简介

- 添加图书
- 删除图书
- 查询图书
- 显示全部图书
- 添加用户
- 删除用户
- 显示全部用户
- 借阅图书
- 归还图书
- 查看用户当前借阅情况

## 系统特点

- 使用面向对象设计
- 图书和用户信息分别使用 `map` 进行管理
- 支持图书借阅历史记录
- 每个用户最多可借阅 5 本图书
- 控制台菜单交互，操作简单

## 运行环境

- C++11 及以上
- 支持标准输入输出的编译器，例如：
  - g++
  - clang++
  - MSVC
